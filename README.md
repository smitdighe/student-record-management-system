# 🎓 Student Record Management System 

A simple yet powerful **menu-driven CLI application** written in C to manage student records using file handling.

---

## 🚀 Features

* ➕ Add Student
* 📋 View All Students
* 🔍 Search Student by ID
* ✏️ Update Student Details
* ❌ Delete Student
* 📊 Sort Students (by marks/name)
* 🏆 Display Topper
* 🧠 Prevent Duplicate IDs

---

## 🛠️ Tech Stack

* Language: C
* Concepts Used:

  * Structures
  * File Handling (Text/Binary)
  * Functions
  * CLI Menu System

---

## ⚙️ How to Run

```bash
gcc main.c -o app
./app
```

---

## 🖥️ Example Usage

```
===== Student Management System =====
1. Add Student
2. View Students
3. Search Student
4. Update Student
5. Delete Student
6. Exit
```
